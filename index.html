Introdução

O perfil de interoperabilidade da Saúde iD visa habilitar troca de dados de paciente através da definição de padrões com as características e contextos dos pacientes da Saúde iD. Tem como objetivo também unificar as várias fontes e visões de paciente existentes no contexto da Saúde iD adotando uma terminologia padrão e facilitando a integração e desacoplamento entre eles.

O guia de implementação Saúde iD é baseado no FHIR Versão R4 e tem como plano futuro utilizar o Br-Core como base assim que ele for publicado oficialmente.

Como utilizar este guia?

Este guia é dividido em páginas que são listadas no topo na barra de menus.

  • Início: Provê uma introdução e ambientação para utilizar este guia de implementação.
  • Orientações gerais: Provê orientação, definições e requisitos de conformidade comuns a todos os atores previstos neste guia.
  • Artefatos FHIR: Provê uma descrição detalhada e formal de todos os objetos e entidades FHIR deste guia.
    • Perfis: Essa página lista todos os perfis utilizados por este guia. Cada perfil inclui uma descrição narrativa, definição formal e um guia de início rápido para cada operação.
  • Downloads: Provê links para download de cada artefato definido.
  • Histórico de mudanças: Essa página provê o histórico de alterações e versões deste guia.
